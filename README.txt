Пояснительная записка (README.txt) для сайта

Данный файл README предназначен для описания процесса запуска проекта на Vue и использования базы данных, расположенной в файле database.sql в папке src/api.
Запуск проекта на Vue

Для успешного запуска проекта на Vue следуйте инструкциям ниже:
Предварительные требования

    Убедитесь, что на вашем компьютере установлен Node.js. Вы можете скачать его с официального сайта https://nodejs.org и установить, следуя инструкциям.
    Установите Vue CLI, выполнив следующую команду в командной строке:

npm install -g @vue/cli


Установка и настройка проекта

    Склонируйте репозиторий проекта или скачайте его в виде ZIP-архива и распакуйте.
    Откройте командную строку и перейдите в корневую папку проекта.

Установка зависимостей

    Выполните команду для установки зависимостей проекта:

npm install


Запуск проекта

    После успешной установки зависимостей, выполните следующую команду для запуска проекта:

npm run serve


    Откройте веб-браузер и перейдите по адресу http://localhost:8080 для просмотра сайта гостиницы.

Использование базы данных

База данных для проекта находится в файле Practic.sql в папке src/api. Для использования этой базы данных, выполните следующие шаги:

    Убедитесь, что у вас установлена и запущена СУБД (система управления базами данных) MySQL или любая другая, совместимая с MySQL.
    Создайте новую базу данных в вашей СУБД и дайте ей имя, например, hotel_db.
    Импортируйте содержимое файла database.sql в созданную базу данных. Это можно сделать с помощью утилиты командной строки или с помощью графического интерфейса управления базами данных, такого как phpMyAdmin или MySQL Workbench.
    Убедитесь, что настройки подключения к базе данных в файле проекта src/api/db.js соответствуют вашей конфигурации СУБД:

javascript

{
    private $host = 'localhost'; // хост базы данных
    private $user = 'root';  // имя пользователя базы данных
    private $password = ''; // пароль пользователя базы данных
    private $database = 'Practic'; // имя созданной базы данных
}

    Теперь база данных готова к использованию в проекте.

Если вы следуете указанным выше инструкциям, вы сможете успешно запустить проект на Vue и использовать базу данных для вашего сайта.
